# Portfolio - Flexbox

## Autor
Antonio Padín Deibe

## Descripción
Este proyecto es un portfolio personal desarrollado con HTML y CSS usando Flexbox.  
Incluye un sidebar con foto de perfil, enlaces a redes, secciones de formación, experiencia, habilidades y proyectos.  
El diseño es responsive, con tarjetas y barra de progreso adaptables a dispositivos móviles.

## Tecnologías utilizadas
- HTML5
- CSS3 (Flexbox, Gradientes, Box-shadow, Backdrop-filter)
